# VoiceSharer 音频共享软件

VoiceSharer 是一款高效的 Windows 音频分发工具，允许您将电脑声音同时输出到多个物理设备（如蓝牙耳机、音箱等），并支持为每个设备独立设置延迟补偿，解决音画同步问题。

## 🛠 准备工作（必读）

由于 Windows 系统底层限制，Python 无法直接创建虚拟声卡驱动。在使用本软件前，您**必须**安装第三方虚拟音频驱动：

1.  **下载并安装 VB-Audio CABLE**：
    *   推荐：[VB-Audio CABLE](https://vb-audio.com/Cable/) (免费)。
2.  **重启电脑**：
    *   安装完成后，**请务必重启您的计算机**，否则驱动无法生效。
3.  **设置默认播放设备**：
    *   在 Windows 声音设置中，将 **"CABLE Input (VB-Audio Virtual Cable)"** 设置为您的**默认播放设备**。
4.  **运行 VoiceSharer**：
    *   启动软件后，它将监听 "CABLE Output" 并将音频分发到您勾选的所有物理设备。

## 📦 安装说明

如果您是从源码运行，请确保已安装 Python 3.8+ 环境，并执行以下命令安装依赖：

```bash
pip install -r requirements.txt
```

## 🚀 使用指南

1.  **启动软件**：
    ```bash
    python main.py
    ```
2.  **选择设备**：
    *   在左侧设备列表中，勾选您想要同时输出声音的设备（如：您的耳机和音箱）。
3.  **调整延迟**：
    *   如果发现蓝牙耳机声音慢于音箱，请调节该设备的“延迟（毫秒）”滑块，直到声音同步。
4.  **方案管理**：
    *   点击“保存为新组合”，可以快速记录当前的设备勾选状态，方便下次一键切换。
5.  **自动化设置**：
    *   勾选“开机自启动”和“最小化至托盘”，让软件在后台为您提供无感音频共享服务。
