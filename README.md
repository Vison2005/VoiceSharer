# VoiceSharer

VoiceSharer 是一个基于 Python 和 PyQt5 的多设备音频同步工具。它可以将一个音频源实时同步到多个输出设备，并提供高精度的延迟校准。

## 核心功能

- **多设备同步**：支持同时向多个音频设备输出声音。
- **独立控制**：每个设备拥有独立的音量调节和延迟补偿。
- **自动校准**：通过麦克风反馈和扫频信号 (Chirp) 自动对齐设备延迟。
- **设备组合**：支持保存常用的设备勾选组合。
- **现代化 UI**：支持深浅模式切换及平滑过渡动画。
- **便捷特性**：支持开机自启、最小化至系统托盘。

## 技术栈

- **Python 3.10+**
- **PyQt5**: 界面框架
- **SoundDevice / PortAudio**: 音频流处理
- **NumPy**: 音频信号处理与校准计算

## 快速开始

1. **设置音频源**：建议配合 VB-Audio Virtual Cable 使用。
2. **选择输出设备**：在列表中勾选播放设备。
3. **自动校准**：点击“自动同步校准”，跟随提示完成延迟对齐。
4. **手动微调**：点击设备项，在右侧面板调整音量或延迟。

## 打包

```bash
python -m PyInstaller VoiceSharer.spec --noconfirm
```
可执行文件位于 `dist` 目录。
